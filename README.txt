FANNIE - backend environment for IS4C

This is an unstable distribution of of the FANNIE code.  Do NOT run this software in production without THOROUGHLY testing EVERY script that you intend to run.

The core functionality is solid, this includes:
ITEM MAINTENANCE
SALES BATCHING
BASIC REPORTING
SYNCHRONIZING WITH LANES
MEMBER-MGMT

In addition to the core features, other modules ARE included with this distribution.  These are to aid in the development of your own modules or to offer a starting point.
